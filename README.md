<h1 class="code-line" data-line-start=0 data-line-end=1 ><a id="Asteroid"></a>Asteroid</h1>

<p class="has-line-data" data-line-start="3" data-line-end="4"><img width="100%" height="100%" src="https://i.ibb.co/yg8k3Lw/screencapture-localhost-5173-2023-05-31-20-20-05.png" alt="N|Solid"></p>

<p class="has-line-data" data-line-start="5" data-line-end="6">Asteroid is a content-driven platform that generate articles about Space and Astronomy, All the Articles and blogs written by ChatGPT which is an artificial intelligence chatbot developed by OpenAI. also Strapi used as a backend for the project with SQLite database.</p>
